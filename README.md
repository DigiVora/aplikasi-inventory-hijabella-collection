# 📦 Aplikasi Inventory Hijabella Collection

Aplikasi Inventory Hijabella Collection adalah sistem informasi persediaan barang berbasis Java NetBeans dan MySQL yang digunakan untuk mengelola data barang, barang masuk, barang keluar, serta laporan persediaan secara efektif dan efisien.

## 🎯 Tujuan Pengembangan

Aplikasi ini dibuat untuk memenuhi tugas mata kuliah Pemrograman Lanjutan serta membantu proses pengelolaan persediaan barang pada UMKM Hijabella Collection.

---

## 🚀 Cara Menjalankan

1. Clone repository

```bash
git clone https://github.com/DigiVora/aplikasi-inventory-hijabella-collection.git
```

2. Import project ke Apache NetBeans.

3. Buat database MySQL.

4. Import file database yang tersedia.

5. Sesuaikan konfigurasi koneksi database.

6. Jalankan project.
